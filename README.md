# Telecom-BPO-Revenue-collections-Analytics

## Project Overview

This project demonstrates a SQL Server-based Invoice & Payment Management System for a telecom company.

The system helps the Finance team track customer invoices, payments, outstanding balances, and overdue invoices.

---

## Business Problem

The finance department maintains invoice and payment records manually, making it difficult to:

- Track outstanding invoices
- Monitor customer payments
- Identify overdue invoices
- Prepare monthly finance reports

---

## Project Objectives

- Store customer information
- Generate invoices
- Record payments
- Calculate outstanding balances
- Track overdue invoices
- Build finance reports using SQL
